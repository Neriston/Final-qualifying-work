Данный репозиторий содержит материалы и программную реализацию выпускной квалификационной работы (ВКР) на тему: «Разработка и сравнение методов оценки опционов азиатского типа».

В рамках работы предлагается разработать и сравнить методы оценки азиатских опционов с использованием численных подходов: метода Монте-Карло и конечно-разностных схем в расширенных моделях ценообразования — стохастической волатильности Хестона и диффузионно-прыжковом процессе Мертона; далее провести сравнение полученных теоретических цен с рыночными оценками на основе исторических данных. Основная цель работы — выявление наиболее адекватной модели для описания рыночных реалий при оценке экзотических опционов азиатского типа.
